[[section-glossary]]
== Glossary

[cols="1,4"]
|===
| **Term** | **Definition**

| **Channel**
| A cryptographic group context. Participants share keys to read/write.

| **DHT**
| Distributed Hash Table. The decentralized storage for finding peers and OHDescriptors.

| **Flaschenpost**
| (Message in a Bottle) The metaphor for a Store-and-Forward packet in Redpanda.

| **Full Node**
| A server that routes packets and stores DHT records.

| **Garlic Message**
| A bundled, multi-layer encrypted message format (inspired by I2P).

| **Light Client**
| The mobile application running on end-user devices.

| **Outbound Handle (OH)**
| A temporary address (mailbox) on a Full Node where a client receives messages.

| **OHDescriptor**
| The public record containing the OH address and public keys, published in the DHT.

| **Reverse Garlic**
| A technique where a sender uses a pre-built "Reply Block" to send data back to a receiver without knowing the receiver's address.

| **R-ACK** (Routing ACK)
| An acknowledgement generated by the network (Outbound Node) to confirm delivery to the mailbox.

| **Channel-ACK**
| An acknowledgement generated by the User's Client to confirm reading/processing.

|===
